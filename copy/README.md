## Resumen del tono

- **Esencia:** El amigo que cocina bien y te salva la semana
- **Estilo:** Casual, directo, sin corporativo BS
- **Diferencial en copy:** Fresco (nunca congelado), app real, simple

## Reglas aplicadas

✓ Español argentino
✓ Sin tildes en UI
✓ Sin "revolucionario", "disruptivo", etc.
✓ Foco en: fresco, real, simple, saludable
