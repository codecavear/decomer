# DeComer

Viandas saludables por suscripción en Córdoba.

## Stack

- **Nuxt 4** + **Nuxt UI 4** + **TailwindCSS**
- **Drizzle ORM** + **PostgreSQL**
- **MercadoPago** (pagos)
- **Railway** (deploy)

## Setup

```bash
bun install
bun run dev
```

## Features (MVP)

- Catálogo con fotos, ingredientes, macros/calorías
- Suscripción semanal/mensual + pedido único
- Filtros: veggie, sin gluten, low carb, proteico
- Pago: MercadoPago, tarjeta, transferencia
- Perfil: alergias, preferencias, dirección, horario
- Push notifications + magic link login
- Rewards/puntos + referral
